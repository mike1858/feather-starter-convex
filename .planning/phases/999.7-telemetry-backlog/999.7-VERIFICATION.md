### Gaps Summary

Two gaps block full goal achievement:

**Gap 1 — Sentry package not installed:** `@sentry/react` and `@sentry/vite-plugin` are declared in `package.json` and committed to `package-lock.json` (494-line diff in commit `848b386`) but `npm install` was never run to actually install them. `node_modules/@sentry` does not exist. This causes `src/shared/__tests__/error-capture.test.ts` to fail entirely on import (Vite cannot resolve the dynamic `import("@sentry/react")` in error-capture.ts even for `vi.mock` tests). Fix: run `npm install`.

**Gap 2 — Registry field regression:** Plan 01 commit `424ae10` added the `telemetry` and `errorReporting` sections to `projectYamlSchema` but accidentally omitted the `registry` field that was added by phase 999.6 (commit `83b1bbe`). The git diff confirms `- registry: z` was removed. This causes 2 test failures in `bin/__tests__/registry.test.ts`. Fix: re-add `registry: z.object({ url: z.string().url() }).optional()` to `projectYamlSchema`.

Both are simple mechanical fixes: one `npm install`, one line of schema code restored.
