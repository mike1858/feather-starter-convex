## Deviations from Plan

**[Rule 1 - Bug] Zod v4 default cascade behavior:** `.default({})` on z.object does not propagate inner field defaults. Fixed by providing full default objects. Affects operations schema.

**[Rule 1 - Bug] z.record with z.enum key in Zod v4:** When partial record provided, missing keys fail validation against the enum key type. Fixed access.permissions to use z.object() with per-field defaults.

**[Rule 3 - Blocking] Cross-entity validation scope:** detailView.relatedRecords references entities from separate YAML files (subtasks, workLogs). Single-file validation cannot verify these. Moved to pipeline-level validation where all YAMLs are loaded.
