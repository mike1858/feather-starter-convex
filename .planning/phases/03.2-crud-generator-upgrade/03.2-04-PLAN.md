    **`thumbnail`** — small image left:
    - Content: `flex items-start gap-4`

      - Text section: same as `none` but right-aligned

    **`cover`** — large image top:
    - Wrapper: `rounded-lg border border-border bg-card overflow-hidden shadow-card ...`
    - Image: `aspect-video w-full object-cover bg-muted` with gradient placeholder fallback
    - Content below image: same as `none`

    **`overlay`** — image fills card, text overlaid:
    - Wrapper: `rounded-lg overflow-hidden shadow-card ...`
    - Image: `absolute inset-0 w-full h-full object-cover bg-muted`
    - Gradient: `absolute inset-0 bg-gradient-to-t from-black/70 via-black/20 to-transparent`
    - Content: `relative z-10 flex flex-col justify-end h-full {size-padding}`
    - Text: `text-white`, status badge text `text-white/90`
    - Min height: `aspect-[4/3]` (compact) or `aspect-[3/2]` (default/large)

    MANDATORY card hover effect on all variants: `shadow-card hover:shadow-card-hover hover:-translate-y-px transition-all duration-200 ease-out`

    - `string`: `text-sm text-primary`, first column `font-semibold`

    **Column width** (from YAML per-column `width` config):
    - `auto` (default): no explicit width
    - `xs`: 80px, `sm`: 120px, `md`: 200px, `lg`: 300px, `xl`: 400px
    - Applied via inline style `style={{ width: '{pixels}px', minWidth: '{pixels}px' }}`

    **Design variants** from `views.table.design`:
    - `minimal` (default): `hover:bg-muted/30 transition-colors border-b border-border`
    - `striped`: add `even:bg-surface-secondary`

    **Column sorting:**
    - Sortable headers: `cursor-pointer hover:text-primary transition-colors`
    - Click cycle: none → ascending → descending → none
    - Active sort: `ChevronUp` (asc) or `ChevronDown` (desc) `h-3 w-3 inline ml-1 text-primary`
    - State: `const [sortColumn, setSortColumn] = useState<string | null>(null);` and `const [sortDirection, setSortDirection] = useState<"asc" | "desc">("asc");`
    - Client-side sorting (sort data array before render)

    **Row grouping** (if `views.table.groupBy` is set):

    - State: `const [collapsedGroups, setCollapsedGroups] = useState<Set<string>>(new Set());`
