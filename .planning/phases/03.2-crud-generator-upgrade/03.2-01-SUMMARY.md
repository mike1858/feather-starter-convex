## Decisions Made
- Generator utilities written as plain ESM .js so Plop can import them directly without TypeScript compilation step. types.ts remains TypeScript for JSDoc type imports.
- deepmerge configured with array replacement (not merge) so feature YAML arrays completely override defaults rather than concatenating.
- Enum color palette uses 8-slot round-robin by declaration order with per-field override support via colors: map in YAML.
- JSDoc comment containing JSX-like syntax caused Vite parse failure; simplified comment to avoid the pattern.
