## Decisions Made
- Used double-stache `{{}}` instead of triple-stache `{{{}}}: TypeScript identifiers never contain HTML entities, making escaping irrelevant for .ts templates
- Schema template builds Zod types inline rather than calling the `zodType` helper, because the schema needs precise control over exports, max-length constants, and the required/optional distinction at the field level
- Enum fields with `transitions` are excluded from `createInput` (use the default first value) and get a dedicated `updateStatus` mutation instead
