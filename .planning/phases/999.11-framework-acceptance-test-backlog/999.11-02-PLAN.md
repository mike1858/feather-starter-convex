## Context

Three example apps (todos, tickets, contacts) are fully installed with working frontend + Convex backend. No E2E tests exist for any of them. All follow an identical generated pattern: inline form (placeholder input + "Add" button), list view, item with hover-reveal delete using double-check "Delete?" confirmation.

**Existing patterns to follow:**
- `e2e/tasks.spec.ts` — CRUD test pattern (create, edit, status transitions, delete)
- `e2e/helpers.ts` — `createTask(session, title)` pattern: `fillIn(placeholder).clickButton("Add").assertText(title)`
- `e2e/fixtures.ts` — `createConvexTest` fixture with `clearAll` auto-cleanup
- Delete pattern: find row, click trash icon, see "Delete?" text, click again

**Critical detail — delete confirmation text:**
- Tasks (hand-crafted): "Are you sure?"
- Generated features (todos, tickets, contacts): "Delete?"

**UI details per app:**
- Todos: placeholder "Add todo...", button "Add". No badges.
- Tickets: placeholder "Add ticket...", button "Add". Shows "open" status badge + "low" priority badge after creation.
- Contacts: placeholder "Add contact...", button "Add". Shows "lead" status badge after creation.

**BLOCKER:** `convex/testing/clearAll.ts` is missing `todos`, `tickets`, `contacts` tables. Must be fixed first or E2E tests will have leftover data between runs.

- [ ] **Task 2.2:** Create E2E spec files for todos, tickets, and contacts

  Create three spec files following the `e2e/tasks.spec.ts` pattern. Each spec: sign up a fresh user, navigate to the feature page, create an item, verify it appears, delete it with double-check, verify it is gone.

  **`e2e/todos.spec.ts`:**
  ```typescript
  import { test, expect } from "./fixtures";
  import { signUp, createTodo, uniqueEmail } from "./helpers";

  test.describe("Todo flows", () => {
    test("create and delete a todo", async ({ session, page }) => {
      const email = uniqueEmail();
      await signUp(session, email, "password123", "todouser1");

      await session.visit("/dashboard/todos");
      await session.assertText("Todos");

      // Create a todo
      await createTodo(session, "Test Todo E2E");

      // Delete: find the item row, click trash, confirm "Delete?"
      const row = page.locator("div", { hasText: "Test Todo E2E" }).first();
      const deleteBtn = row.getByRole("button", { name: /trash|delete/i }).or(
        row.locator("button:has(svg.lucide-trash-2)"),
      );
      await deleteBtn.first().click();

      // Double-check confirmation
      await session.assertText("Delete?");
      await page.getByText("Delete?").click();

      // Verify removal
      await session.refuteText("Test Todo E2E");
    });
  });
  ```

  **`e2e/tickets.spec.ts`:**
