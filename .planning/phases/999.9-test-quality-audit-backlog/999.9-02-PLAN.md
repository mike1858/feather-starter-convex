# Plan 999.9-02: v8 Ignore Purge + Coverage Exclusion Trim + New Tests

## Objective

Blanket-remove all 154 `v8 ignore` directives (D-04), delete dead code discovered during the purge (D-05), trim illegitimate coverage exclusions from `vitest.config.ts`, and write tests for all newly exposed code paths. Target: `npm test` passes at 100% coverage with zero `v8 ignore` directives remaining.

## Why This Matters

The current 100% coverage is achieved partly through 76 ignore blocks and 60 exclusion entries. This plan replaces ignores with real tests, making the coverage number honest. Dead code gets deleted rather than hidden.

## must_haves
- [ ] Zero `v8 ignore` directives in the codebase (excluding `.worktrees/` and `node_modules/`)
- [ ] `vitest.config.ts` exclusions trimmed — only D-03 legitimate exclusions remain
- [ ] Dead code identified in findings report deleted (not tested)
- [ ] `npm test` passes with 100% coverage on all thresholds
- [ ] No new `v8 ignore` directives introduced
