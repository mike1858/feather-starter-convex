<task type="auto">
  <name>Task 1: Install Lefthook and configure pre-commit hooks</name>
  <files>lefthook.yml, package.json</files>
  <action>
    1. Install lefthook: `npm install -D lefthook`

    2. Create `lefthook.yml` at project root:
       ```yaml
       pre-commit:
         parallel: false
         commands:
           typecheck:
             run: npx tsc -p tsconfig.app.json --noEmit && npx tsc -p tsconfig.node.json --noEmit && npx tsc -p convex/tsconfig.json --noEmit
           test-coverage:
             run: npx vitest run --coverage
       ```
       - `parallel: false` because typecheck should pass before running tests (faster failure)
       - Both commands must pass for commit to proceed

    3. Install git hooks: `npx lefthook install`
       - This writes to `.git/hooks/pre-commit`
       - Lefthook binary is in node_modules, no global install needed

    4. Verify the hook is installed: `ls -la .git/hooks/pre-commit`
  </action>
  <verify>
    <automated>cat lefthook.yml && ls .git/hooks/pre-commit</automated>
  </verify>
