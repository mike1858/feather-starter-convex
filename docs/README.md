# Feather Starter Documentation

## Quick Start

```sh
# Install dependencies
npm install

# Set up a new Convex project
npx convex dev --configure=new --once

# Set up Convex Auth
npx @convex-dev/auth
```

See the [main README](../README.md) for environment variable setup and more details.
